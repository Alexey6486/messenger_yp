## Установка

- `npm install` — установка зависимостей,
- `npm run start` — запуск приложения,
- `npm run build` — сборка стабильной версии,
- `npm run lint` — запуск проверки eslint,
- `npm run lint:style` — запуск проверки стилей stylelint.
